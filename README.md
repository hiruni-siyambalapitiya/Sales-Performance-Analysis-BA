# Retail Sales Performance Analysis & BI Dashboard

## Project Overview
This project demonstrates an end-to-end Business Analyst and Business Intelligence solution for a retail company facing delays in manual Excel reporting.

## Business Problem
Sales data was tracked using manual Excel reports, causing delayed insights and inefficient decision-making.

## Objectives
- Improve sales visibility
- Reduce manual reporting
- Enable faster management decisions

## Tools Used
- Excel
- Power BI
- Draw.io
- Microsoft Word

## Key Deliverables
- Problem Statement & Stakeholder Analysis
- Business & Functional Requirements
- Agile User Stories & Backlog
- As-Is & To-Be Process Diagrams
- System Context Diagram
- Power BI Interactive Dashboard

## Dashboard Highlights
- Total Sales KPI
- Monthly Sales Trend
- Sales by Region & Product
- Donut Chart for Sales Contribution
